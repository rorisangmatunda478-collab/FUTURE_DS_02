# 🧠 Customer Churn & Retention Analysis
This project analyzes customer behavior to understand churn patterns, retention drivers, and engagement trends using interactive dashboards built in Tableau.
🎯 Objective
The goal of this project is to:
Identify key factors influencing customer churn
Analyze engagement and retention relationships
Provide actionable insights to improve customer retention

# 📁 Dataset Overview
The dataset consists of customer-level data including:
Customer usage duration
Tenure (months)
Subscription plan tiers (Basic, Enterprise, Pro)
Industry segments (HealthTech, FinTech, EdTech, DevTools, Cybersecurity)
Churn status (True/False)

# 🛠️ Tools & Technologies
-Tableau Public – Data visualization & dashboarding
-SQL (via Jupyter/Kaggle) – Data cleaning and preprocessing
-Excel/CSV – Raw dataset format 
​-Data Cleaning & Preprocessing: Python (Pandas/NumPy) via Kaggle.
​-Data Integration: Merged 4 disparate raw data files into a single unified source.
-​Calculations: Custom Calculated Fields for Churn Rate, Power User Ratios, and Average Tenure.

# 📊 Dashboard Features
KPI Summary
Total Customers: 500
Churn Rate: 22%
Usage Ratio: 98.8%
Average Tenure: 2.8 months
Customer Engagement vs Retention
Scatter plot showing relationship between usage and churn behavior
Churn by Plan Tier
Comparison of churn rates across subscription tiers
Churn by Industry
Industry-level breakdown of churn distribution
Customer Retention Over Time
Trend analysis of churn behavior across tenure

# 💡 Key Insights
Customers with lower usage duration are significantly more likely to churn
Churn rate is consistent across plan tiers (~22%), suggesting pricing alone is not the issue
Short tenure customers (early lifecycle) show higher churn risk
Industry differences exist, but churn patterns remain relatively similar across segments
High overall usage ratio (98.8%) suggests engagement exists, but retention strategies may be weak 

# 🚀 Recommendations
-Improve onboarding experience to reduce early churn
-Implement engagement-based retention strategies (e.g., usage nudges, reminders)
-Introduce customer success touchpoints within the first 3 months
-Develop personalized retention campaigns based on usage behavior
-Monitor low-engagement users proactively
